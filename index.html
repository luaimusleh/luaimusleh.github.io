Final SQL Project

Objective: Developed a database project involving schema design, relational diagrams, and comprehensive queries to analyze and manage client-consultant relationships and contract data.

View Project Highlights ▼
  • Schema Design: Created a 3NF-compliant schema with the following tables:
    • Region: Tracks location and regional associations.
    • Contract: Stores contract details, including status and difficulty.
    • Client: Manages client-specific data such as personal details and location.
    • Consultant: Tracks consultant details, expertise, and assignments.
    • Cross-Referencing: Includes ContractConsultant, Class, and ClassConsultant tables.
  • Query Development: Designed 10 optimized SQL queries, including:
    • Identifying client-consultant relationships by region.
    • Tracking active contracts and filtering by difficulty levels.
    • Displaying client and contract details for performance analysis.
  • Relational Diagram: Visualized database relationships through an ERD and relational schema for seamless data flow.
